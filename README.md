## Requirements

* npm
* MongoDB
* Redis

## Installing

Install dependencies 
```
npm install
```

Run MongoDB and Redis locally
```
docker-compose up -d
```

## Executing 

Run application with
```
npm run dev
```

Redis commander: http://127.0.0.1:8081
