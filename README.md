## Requirements

- Node.js
- npm
- MongoDB (I use Docker container for local development)
- Redis (I use Docker container for local development)

## Installing

Install dependencies

```
npm install
```

Run MongoDB and Redis locally

```
docker-compose up -d
```

## Executing

Run application with

```
npm run dev
```
